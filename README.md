# House Price Prediction Using Machine Learning

## Problem Statement

The objective of this project is to build a machine learning model that can predict house prices based on various housing features such as area, bedrooms, bathrooms, parking facilities, and other amenities.

## Dataset

Housing Dataset from Kaggle.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib
* Jupyter Notebook

## Methodology

### Data Preprocessing

* Checked for missing values
* Converted categorical variables into numerical values
* Prepared dataset for machine learning

### Feature Selection

Selected housing attributes as input features and house price as the target variable.

### Train-Test Split

Split the dataset into training and testing sets.

### Model Training

Used Random Forest Regressor to train the model.

### Model Evaluation

Performance was evaluated using:

* Mean Absolute Error (MAE)
* R² Score

## Results

* MAE: 1024081.03
* R² Score: 0.6128

The model successfully learned patterns from housing features and generated price predictions for unseen houses.
